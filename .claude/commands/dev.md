Start the development server for StopShorts.

## Command

```bash
npx expo start
```

## Options

- Press `a` to open Android emulator
- Press `r` to reload
- Press `m` to toggle menu

## With Android Device

```bash
npx expo start --android
```

## Clear Cache

If experiencing issues:
```bash
npx expo start --clear
```
