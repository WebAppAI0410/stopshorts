# StopShorts - 料金体系 & マネタイズ設計

## 1. マネタイズ戦略概要

### 1.1 基本方針

**「コミットメントデバイスとしての前払い制」**

- 1日間のフル体験（無料）で価値を実感
- 体験後は「課金して本格利用」or「機能制限版で継続」を選択
- 前払い制により心理的コミットメントを強化

### 1.2 App Store規約対応

> **App Review Guidelines 4.10**
> "You may not monetize built-in capabilities provided by the hardware or operating system, such as ... Screen Time APIs..."

**対応策**:

| 無料で提供（必須） | 有料で提供（OK） |
|------------------|----------------|
| 基本的な遮断機能（1日1回） | **継続的な遮断**（何度でも介入） |
| 5分固定、固定文言 | 時間カスタマイズ、パーソナライズ文言 |
| 対象アプリ1つ | 対象アプリ無制限 |
| 今日の記録のみ | 詳細統計・分析・レポート |
| - | 行動変容コーチング |

**核心**: Screen Time APIによる「遮断機能」自体は無料で提供。課金対象は「継続的な介入」「コーチング」「パーソナライズ」という**行動変容プログラム**。

```

「5分経過しました」
  ↓
閉じる or 続ける
  ↓

  ↓
翌日リセット

→ iOS純正Screen Timeとほぼ同等
→ 「本気で止めたい」人には物足りない
```

**有料版**:
```
TikTok起動
  ↓
5分経過 → Shield表示（パーソナライズ文言）
「質の良い睡眠は明日の自分への投資です」

  ↓
閉じる → 「素晴らしい！今日3回目の成功です」
  ↓
続ける → さらに5分後に再度Shield
  ↓
何度でも介入（本気で止めさせる）
  ↓
週次レポート、ストリーク表示、詳細分析

→ 継続的な介入で本当に習慣を変える
```

## 9. 審査対応

### 9.1 レビューノートでの説明

```
【課金について】

StopShortsの課金は、以下の付加価値サービスに対するものです:

1. 継続的な行動介入
   - 機能制限版: 1日1回の遮断
   - 有料版: 何度でも継続的に介入
   → これは「より効果的な行動変容プログラム」への課金であり、
     Screen Time API機能そのものへの課金ではありません

2. 行動変容コーチング
   - パーソナライズされたメッセージ
   - 目的別・時間帯別の文言
   - 週次フィードバック
   - 失敗時の回復サポート

3. 詳細分析・統計
   - 週間/月間データの保存・表示
   - 節約時間の計算・可視化
   - 達成レポートの生成

【重要】
Screen Time APIによる基本的な遮断機能は、
機能制限版として全ユーザーに無料で提供しています。
機能制限版でも1日1回の遮断が動作し、
iOS純正Screen Timeと同等の体験が可能です。
```

### 9.2 FAQ（レビュー対応用）

**Q: 無料で遮断機能は使えますか？**
A: はい、機能制限版で1日1回の遮断機能を無料でご利用いただけます。これはiOS純正のScreen Timeと同等の機能です。

**Q: 有料プランと無料プランの違いは？**
A: 有料プランでは、継続的な遮断（何度でも介入）、パーソナライズされたコーチング、詳細統計など、行動変容をより効果的にするための付加価値機能を提供しています。

**Q: なぜ遮断頻度に差があるのですか？**
A: 継続的な介入は、行動変容の研究に基づいた「コーチングプログラム」の一部です。単発の遮断（機能制限版）と、継続的な介入による習慣形成サポート（有料版）は、提供する価値が異なります。
